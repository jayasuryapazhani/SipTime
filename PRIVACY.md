# Privacy Policy – SipTime

**Effective date:** July 18, 2026  
**Last updated:** July 18, 2026

SipTime is a browser extension and companion website designed to provide hydration reminders. This privacy policy explains what information SipTime processes, where it is stored, and which third-party services may receive limited technical data when you use the website.

## 1. Scope

This policy applies to:

- The SipTime browser extension
- The SipTime website at [https://siptime.vercel.app/](https://siptime.vercel.app/)

## 2. Browser Extension Data Practices

### Information SipTime Does Not Collect

The SipTime extension does not collect, sell, share, or intentionally transmit:

- Names, email addresses, or other personal identifiers
- Health or medical information
- Financial or payment information
- Precise or approximate location
- Browsing history
- Website content
- Search history
- Authentication credentials
- Advertising identifiers
- Usage analytics

SipTime does not use content scripts and does not request access to the websites you visit.

### Information Stored by the Extension

SipTime stores only the information required to operate hydration reminders, including:

- Whether reminders are enabled
- Reminder mode
- Reminder interval
- Fixed reminder times
- Quiet-hours settings
- Snooze duration
- The next scheduled reminder time

This information is stored using `chrome.storage.sync`.

When browser synchronization is enabled, the browser may synchronize this information across the user's signed-in browser profiles or devices. This synchronization is provided and controlled by the browser vendor. SipTime does not operate a server that receives, stores, or processes these settings.

Users can remove stored extension data by uninstalling SipTime or clearing the extension's storage through their browser.

## 3. Extension Permissions

SipTime requests only the permissions required for its core functionality:

- `alarms` — schedules hydration reminders
- `notifications` — displays hydration reminders and notification actions
- `storage` — saves reminder preferences and next-reminder information

SipTime does not request host permissions or access to webpage content, browser tabs, browsing history, location, camera, or microphone.

## 4. Notifications

SipTime creates local browser notifications according to the user's saved schedule.

Notification content may include:

- A hydration reminder message
- A Done action
- A Snooze action using the configured snooze duration

These notifications are generated by the browser extension and displayed through the browser and operating-system notification services.

## 5. Remote Code

The SipTime extension does not download or execute remote JavaScript, WebAssembly, or other executable code.

All extension functionality is included in the extension package distributed through the Chrome Web Store or generated from this repository.

## 6. Website Data Practices

The SipTime website is a static informational website.

SipTime does not intentionally:

- Require user registration
- Collect form submissions
- Set first-party tracking cookies
- Include advertising
- Include a custom analytics system
- Create user profiles

### Vercel Hosting

The website is hosted by Vercel.

As part of delivering and protecting the website, Vercel may process standard technical request information, such as:

- IP address
- Browser and device information
- Requested URL
- Request timestamp
- Network and security information

This processing is performed by Vercel under its own privacy terms and infrastructure policies.

### Google Fonts

The website currently uses Google Fonts.

When the website loads, the browser may request font resources from Google domains such as `fonts.googleapis.com` and `fonts.gstatic.com`. These requests may include standard network information such as the visitor's IP address, browser information, and requested font resource.

Google processes this information under its own privacy policies.

## 7. Third-Party Sharing

SipTime does not sell personal information.

The extension does not send reminder settings to SipTime-owned servers or third-party advertising, analytics, or data-broker services.

Limited technical information may be processed by:

- The user's browser vendor when `chrome.storage.sync` synchronization is enabled
- Vercel when serving the SipTime website
- Google when serving Google Fonts used by the website

These providers process information according to their own terms and privacy policies.

## 8. Data Retention

SipTime does not maintain a backend database containing extension-user information.

Extension preferences remain in browser storage until they are changed, cleared, or removed through browser controls.

Website-hosting providers may retain technical logs according to their own retention and security policies.

## 9. Data Security

SipTime minimizes data processing by avoiding accounts, backend storage, advertising, and extension analytics.

No method of software distribution, browser storage, synchronization, or internet transmission can be guaranteed to be completely secure. Users should maintain an updated browser and operating system.

## 10. Children's Privacy

SipTime is a general-purpose productivity utility and is not designed to collect personal information from children.

SipTime does not knowingly collect personal information from children.

## 11. User Choices

Users can:

- Disable reminders from the extension popup or settings page
- Change or restore reminder settings
- Clear extension data through browser controls
- Disable browser synchronization
- Uninstall the extension
- Block third-party font requests using browser privacy tools
- Stop using the website

## 12. Changes to This Policy

This policy may be updated when SipTime's functionality, permissions, storage behavior, website services, or legal obligations change.

Material changes will be reflected by updating the effective date or last-updated date at the top of this document.
